# BASH commands

- `pwd` (pass to working directory)

```sh
$ pwd
```
- `ls` (list files and folders)

```sh
$ ls
```

- `mkdir` (make directory)

```sh
$ mkdir partime
```

- `cd` (change directory)

```sh
$ cd parttime
```
- `clear`(clear terminal output, equivalent to CTRL + L)

```sh
$ clear
```

- `touch` (creates an empty file)

```sh
$ touch staff/yoana-padron/git/README.md
```
